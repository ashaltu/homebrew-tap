# Ashaltu Tap

## How do I install these formulae?

`brew install ashaltu/tap/<formula>`

Or `brew tap ashaltu/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "ashaltu/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
